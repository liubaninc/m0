# M0文档

**M0**是磁云唐泉金服自主研发的可扩展、高性能的区块链框架, 是构建超级联盟网络的底层方案。您可以在各类区块链业务场景应用中将**M0**作为区块链基础设施，构建合规的区块链网络。

核心特点

* **高性能**
    * 实现了智能合约的并发执行和验证。
    * POA+BFT算法确保快速共识且绝对一致性。
    * 支持WASM虚拟机

* **更安全**
    * 多私钥保护的账户体系。
    * 支持SM2国密账户

* **多语言开发智能合约**
    * 支持多种语言的虚拟机
    
* **高灵活性**
    * 模块化设计使得用户可以方便选择适合自己业务场景的解决方案。
    * 提供了全面的、高性能的各业务场景的模块化解决方案。
